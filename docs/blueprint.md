# **App Name**: CASHCHA

## Core Features:

- Google Sign-In: Implement secure user authentication with Google Sign-In, including a signup bonus of 100 coins upon first registration.
- CAPTCHA Earning: Enable users to earn coins by solving CAPTCHAs, rewarding 25 coins per CAPTCHA and displaying a full-screen ad after each completion.
- Withdrawal System: Allow users to redeem coins for real money with fixed exchange rates, supporting UPI transfers and Google Play gift cards.
- Withdrawal History: Display all user transactions with real-time status updates (Pending/Success).
- Profile Management: Enable profile page displaying user details, a referral code system, language/theme settings and a logout button.
- Referral Code Generator: Automatically generate and manage referral codes for each user, tracking referrals and applying appropriate bonuses for successful sign-ups using a referral code.
- Email Notification Tool: Integrate an email service as a tool that confirms withdrawal requests and sends updates to users' registered Gmail addresses. This service will send email confirmation and status updates.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to convey trust and security, fitting for a rewards app. 
- Background color: Light blue (#E0F7FA), a desaturated variant of the primary color to ensure a light color scheme.
- Accent color: Green (#90EE90), analogous to the primary color, chosen to highlight success states such as completed CAPTCHAs.
- Body and headline font: 'PT Sans', a sans-serif typeface suitable for both headlines and body text providing a modern, readable look.
- Note: currently only Google Fonts are supported.
- Use flat, minimalist icons for navigation and actions, ensuring clarity and consistency across the app.
- Employ a clean, minimalist design with intuitive navigation using the bottom navigation bar, ensuring ease of use.
- Implement smooth transitions, micro-interactions, and subtle animations to enhance user engagement and provide feedback for actions.
